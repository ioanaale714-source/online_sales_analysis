Proiet Online Sales Analysi
Acest proiect este o aplicatie Python pentru gestionarea produselor si a unui cos de cumparaturi.

Functionalitati:
Gestiune Produse: -Adaugarea si afisarea detaliilor despre produse(nume, pret,cantitate).
Product Manager: -Administrarea inventarului total.
Cos cumparaturi: -Adaugarea produselor in cos si calcularea sumei totale.
